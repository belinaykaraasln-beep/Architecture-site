# Architecture-site

Bu proje, kişisel bir mimarlık portfolyosunu sergilemek için hazırlanmış basit bir web sitesidir. HTML ve CSS kullanılarak geliştirilmiş olup, responsive tasarımı sayesinde farklı cihazlarda sorunsuz çalışır.

---

## Kurulum

Projeyi kullanmak için öncelikle GitHub reposunu bilgisayarınıza klonlayın: 
`git clone https://github.com/belinaykaraasln-beep/Architecture-site.git`
Daha sonra proje dizinine girin: 
`cd Architecture-site`
Siteyi tarayıcıda açmak için `index.html` dosyasını çift tıklayabilir veya terminalden Mac için:
`open index.html`,
Windows için:
`start index.html` 
komutunu kullanabilirsiniz.
